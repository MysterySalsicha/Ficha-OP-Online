## ✨ Funcionalidades

- 📋 **Criação de fichas** operacionais digitais
- 🔄 **Status em tempo real** — aberto, em andamento, concluído
- 🔍 **Busca e filtros** por data, status e responsável
- 📤 **Exportação** de fichas em PDF
- 👥 **Multi-usuário** com controle de acesso
- 📱 **Interface responsiva** para uso em campo

## 💡 Contexto do Projeto

Solução desenvolvida para digitalizar e centralizar o controle de ordens de produção/operação de uma empresa, substituindo planilhas físicas e de Excel por um sistema web acessível de qualquer dispositivo, com histórico e rastreabilidade completos.

## 🛠️ Stack Técnica

| Camada | Tecnologia |
|--------|-----------|
| Frontend | Next.js 14 + TypeScript |
| Estilização | Tailwind CSS |
| Backend | API Routes (Next.js) |
| Deploy | Vercel |

## 🚀 Rodando Localmente

```bash
# Clone o repositório
git clone https://github.com/MysterySalsicha/Ficha-OP-Online.git

# Instale as dependências
cd Ficha-OP-Online
npm install

# Configure as variáveis de ambiente
cp .env.example .env.local

# Rode em desenvolvimento
npm run dev
```

Acesse `http://localhost:3000`
