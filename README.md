## ✨ Destaques

- 🎭 **Temática Imersiva** — UI customizada inspirada em sistemas de RPG.
- ⚙️ **Lógica de Atributos** — Cálculos automáticos dependentes de múltiplas variáveis.
- 🎒 **Gestão de Inventário** — Sistema de adição e remoção de itens com peso/limites.
- 📝 **Edição em Tempo Real** — Interface reativa para mudanças rápidas.
- 🛠️ **Componentização** — Reuso de inputs e modais complexos.

## 💡 Por que este projeto?

Diferente de apps CRUD simples, o **Ficha OP** possui uma lógica de negócio densa. Cada mudança em um atributo (ex: Agilidade) impacta diversos outros campos derivados. O foco aqui foi criar uma interface que suporte grande volume de dados sem perder a usabilidade e a estética do tema.

## 🛠️ Tecnologias

- **React / TypeScript** — Estrutura e segurança lógica.
- **Tailwind CSS** — Estilização temática rica em detalhes.
- **Lucide React** — Ícones que complementam a interface técnica.

## 🚀 Roadmap (Onde este projeto pode chegar)

- [ ] **Dice Roller:** Rolagem de dados integrada com animações 3D.
- [ ] **Modo Mestre:** Visão panorâmica de todas as fichas de uma sala.
- [ ] **Supabase:** Persistência de dados na nuvem e sistema de login.
